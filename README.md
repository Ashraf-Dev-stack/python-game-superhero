# Superpower Chooser
Choose your superpower in this Pythonterminal game.Text appers with a cool typing effect like in the movies.

## How to Play 
Run the file and pick a number: 
1- Flying
2- Invisibility
3-Super Speed

### Run it
python superpower.py

-No external libraries needed
