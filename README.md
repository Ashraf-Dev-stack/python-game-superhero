# Superpower Chooser

Choose your superpower in this Python terminal game.Text appers with a cool typing effect like in the movies.

## How to Play 

Run the file and pick a number: 

1- Flying

2- Invisibility

3-Super Speed

### Run it

python superpower.py

### Built With

-Python 3 

-No external libraries needed

### Why it's cool

-`slow_print`typing effect 

-Beginner friendly code 

-Clean if/else logic
